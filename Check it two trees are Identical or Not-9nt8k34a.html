Check it two trees are Identical or Not

check if one tree left root right is same as the other tree:

if it's identical return TRUE else return FALSE:

appoach: check if root1->val == root2->val && then compare both trees left node && then compare both tree right nodes 

and one base case is that is any one root is NULL so return true if both are NULL else false;

 

Example:5 2 3 -1 6 -1 -1 -1 -1

5 2 3 -1 6 -1 -1 -1 -1

bool identicalTrees(BinaryTreeNode<int>* root1, BinaryTreeNode<int>* root2) {

if(root1==NULL || root2==NULL)

{

return (root1==root2);

}

return (root1->data==root2->data)&& identicalTrees(root1->left,root2->left) && identicalTrees(root1->right,root2->right);
