Maximum Width of Binary Tree(level order)

Maximum Width of Binary Tree:number of nodes between any two nodes.

  • find the first node & last node of that level.(use: Level order Traversal) .
  • do the indexing of each nodes.

Formula:{ (Last index-First index +1)= no of nodes of the last level }

  • for 0-based indexing  

                      i

(2*i+1)                      (2*i+2)

 

  • for 1-based indexing 

                        i

(2*i)                             (2*i+1)

 int widthOfBinaryTree(TreeNode* root) {

       if(!root) return 0;

        int ans=0;

        queue<pair<TreeNode*,int>>q;

        q.push({root,0});

        while(!q.empty())

        {

            int size=q.size();

            int min=q.front().second;

            int last,first;

            for(int i=0;i<size;i++)

            {

                long long cur_id=q.front().second-min;

                TreeNode* node=q.front().first;

                q.pop();

                if(i==0)

                    first=cur_id;

                if(i==size-1) 

                    last=cur_id;

                if(node->left)

                    q.push({node->left,cur_id*2+1});

                if(node->right)

                    q.push({node->right,cur_id*2+2});

            }

            ans=max(ans,last-first+1);

        }

        return ans;

    }
