How many types of database languages are?

DDL--->Data defination language
CREATE, ALTER, DROP, TRUNCATE, RENAME, etc. All these commands are used for updating the data that?s why they are known as Data Definition Language.

DML---->data manipulation language
SELECT, UPDATE, INSERT, DELETE, etc. These commands are used for the manipulation of already updated data that's why they are the part of Data Manipulation Language.

DCL--->data control language
GRANT and REVOKE. These commands are used for giving and removing the user access on the database. So, they are the part of Data Control Language.

TCL--->transaction control language
 e.g., COMMIT, ROLLBACK, and SAVEPOINT. These are the commands used for managing transactions in the database. TCL is used for managing the changes made by DML.
