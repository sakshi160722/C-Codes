Level order (Bfs)Travseral Technique

level wise print like from left -> -> right
-------------->
--------> this way on each level 

steps:1: make one queue and vector
2:push root element to queue 
3; when q! empty loops start and starts and make for loop of particular size of queue 
4:and  push front element of queue on vector and check if q front ->left and right exist or not if they are exist so push to queue.
5:return vector.
