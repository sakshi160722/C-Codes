Maximum Path Sum in Binary Tree

Path : starts from any node A and go to node B that is path AND in the path single node appears only once!
so find the maximum sum of all path between that two elements of that pat

formula: (node->val)+(left,right)


Example :-4 -2 3 -1 -1 1 1 -1 -1 -1 -1
o/p:5

BRUTE: O(n^n)

approach: find all possible paths b/w these two nodes and find maximum out of all but that is so brute takes more time.

 

https://leetcode.com/problems/binary-tree-maximum-path-sum/

OPTIMAL:

int solve(TreeNode<int>* root,int &maxi)

{

if(!root)

{

return 0;

}

int lh=max(0,solve(root->left,maxi));

int rh=max(0,solve(root->right,maxi));

maxi=max(maxi,lh+rh+root->val);

return max(lh,rh)+root->val;

}

int maxPathSum(TreeNode<int>* root)

{

int maxi=INT_MIN;

solve(root,maxi);

return maxi;

}
